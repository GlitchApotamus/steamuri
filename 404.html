Steam URI Redirects

Welcome to the Steam URI Redirector!

This tool allows you to quickly redirect Steam URIs for specific actions or navigation.


How to Use:

  • Get the Steam URI you want to use (e.g., steam://rungameid/730).
  • Strip off steam://
  • Append the remainder of the Steam URI to this website's base URL.

Examples:

  • Open Games Library

These are examples of the game R.E.P.O.:

To start the game, use either of the following

  • Run Game R.E.P.O.
  • Launch Game R.E.P.O.

To join a specific lobby in the game, perhaps one your friend is hosting, use this one

  • Join a Lobby

Make sure you replace the lobbyId and steamId with the proper digits.


Open the constructed URL in your web browser.

The website will process the Steam URI and perform the corresponding action.


Important Notes:

Ensure the Steam URI is valid and correctly formatted.
